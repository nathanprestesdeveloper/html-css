Principais formatações (adicionais da aula anterior)


Texto marcado

Para marcarmos um texto como este, usamos a tag MARK.
Para mudarmos sua cor, usamos a linguagem css (escrita no caderno), para o texto ficar assim!


Texto Grande e Pequeno

Para deixar o texto um pouco maior (logo não será mais usado), usamos a tag BIG.
Para diminuirmos usamos a tag SMALL


Texto Deletado e Inserido

Para deletarmos um texto, usamos a tag DEL.
Para inserirmos um texto, usamos a tag INS.


Texto Sobrescrito e Subscrito

Para sobrescrevermos um texto como em uma equação matemática (X20+3) usamos o par de tags SUP.
Para subescrevermos um texto como em uma equação química (H2O), usamos o par de tags SUB.
