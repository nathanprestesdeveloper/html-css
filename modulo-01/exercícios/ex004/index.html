Testando o favicon

Para usar um favicon podemos utilizar trÊs maneiras:


Iconarchive (icones prontos)
Favicon.cc (fazer a mão o seu ícone)
Favicon.io (Fazer de maneira mais simples ou converter uma imagem pronta)
Agora para podemos colocar basta digitar Link:favicon na linha de código, apagar o que está escrito dentro das aspas e colocar o icone que você escolheu. (não esqueça de colocar dentro de head)

Obrigado pela atenção (não esqueça de olhar o ícone)
