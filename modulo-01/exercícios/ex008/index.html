Tipos de listas

Dentro deste tópico, nós temos dois tipos de listas: Ordenadas e Não ordenadas

Listas Ordenadas

Para as listas ordenadas (dependem de uma ordem), utilizamos o par de tags OL para criar a lista e LI para adicionar itens:

  1. Acordar
  2. Colocar a roupa
  3. Escovar os dentes
  4. Tomar café

Para mais conteúdo, olhe no caderno.

Listas Não Ordenadas

Para a criação de listas não ordenadas (não dependem de uma ordem), utilizamos o par de tags UL para a lista e LI para itens, veja:

  • Comprar manteiga
  • Ligar para o João
  • Varrer a cozinha
  • Limpar a casa

Para mais informações de modificações, consulte o caderno.
